En el archivo queries.sql se encuentran 38 consultas SQL para MySQL diseñadas para respaldar las pruebas funcionales de OrangeHRM Open Source desde la capa de base de datos. Las consultas permiten validar la integridad de los datos, verificar la persistencia de operaciones realizadas desde la interfaz, auditar usuarios y roles, revisar la gestión de empleados y permisos (Leave), detectar inconsistencias y confirmar la existencia de defectos encontrados durante la ejecución de los casos de prueba. Están organizadas en cinco secciones: Autenticación, Gestión de Empleados (PIM), Gestión de Permisos (Leave), Integridad de Datos y Auditoría General, siguiendo un enfoque práctico de QA orientado a la validación posterior a las pruebas funcionales.
